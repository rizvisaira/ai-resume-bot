# Career Me Chatbot

A conversational AI assistant that represents Saira Rizvi's professional background and career information. This chatbot can answer questions about skills, experience, and professional journey using RAG (Retrieval-Augmented Generation) with a vector store of profile documents.

## Features

- Interactive chat interface using Gradio
- Vector-based document retrieval using ChromaDB
- OpenAI GPT-4 for natural language responses
- Professional career and skills information

## Setup

The application will run automatically on Hugging Face Spaces. Make sure to set the following secrets:

- `OPENAI_API_KEY`: Your OpenAI API key

## Usage

Simply start chatting with the bot about career-related questions, skills, experience, or professional background.
